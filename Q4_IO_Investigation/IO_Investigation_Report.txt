4. Resource Limits Observed
Command: ulimit -a, ulimit -n
Output: [paste resource_limits.txt and max_open_files.txt content]
Observation: The max open file descriptor limit affects how many files/sockets a

5. How Linux Manages File I/O
Every open file is represented as a file descriptor mapped to an entry in the 
process's file table, pointing to the actual inode/data. Redirection operators 
reassign what a descriptor points to, without the program needing to know its 
output was redirected.
